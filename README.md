# 📊 SQL Retail Sales Analysis Project

This is an intermediate-level SQL project that simulates an e-commerce retail database and performs business analysis using SQL queries.

## 🚀 Features
- Relational database design (Customers, Orders, Products)
- Data insertion and schema creation
- Analytical queries using:
  - JOINs
  - GROUP BY & Aggregations
  - Subqueries
  - Window Functions

## 📂 Database Schema
- Customers
- Products
- Orders
- Order Details

## 📈 Key Insights
- Top customers by total spending
- Best-selling products
- Monthly revenue trends
- Repeat customers analysis
- Product pricing insights

## 🛠️ Tools Used
- SQL (SQLite / MySQL / PostgreSQL)
- Visual Studio Code

## ▶️ How to Run
1. Open the `.sql` file in VS Code
2. Connect to a database (SQLite recommended)
3. Run:
   - Table creation queries
   - Insert statements
   - Analysis queries
